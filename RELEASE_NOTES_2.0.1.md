## VibeCode Translator 2.0.1 — Glassmorphism UI

### 新功能
- 全新 glassmorphism 視覺風格：設定視窗與解釋氣泡樣式統一
- 共用 `ui_theme.py` 模組，維持 UI 一致性
- 解釋氣泡加大、標題列與關閉按鈕優化

### 安裝（同事推薦）
1. 下載下方 `VibeCode-Translator-2.0.1-macOS-arm64.zip`
2. 解壓 → 拖到「應用程式」→ 雙擊
3. 若提示「無法驗證開發者」→ 系統設定 → 隱私權與安全性 → **仍要開啟**
4. 系統設定 → 輔助使用 → 啟用 **VibeCode Translator**
5. 選單列 ✦ → Settings → 填 API Key
6. 反白 code → `Cmd + Control + E`（或自訂快捷鍵）

### 開發者
```bash
git clone https://github.com/louisty2006/VibeCode_Translator.git
cd VibeCode_Translator
./setup.sh
./run.sh
```
